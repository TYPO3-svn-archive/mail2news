max_attachment_sizeMaximum attachment size in kbInteger, 10-1000
imageextensionsAllowed file extensions for images: Comma separated list of image extensions, like png,jpg,gif. Files with extensions used here will be filtered to news images and will NOT end up at the news file attachmentsString input, advanced
attachmentextensionsAllowed extensions other attachments: Comma separated list of file extensions, like pdf,odt,doc, etc.String input, advanced
category_identifierCategory line identifier: The starting characters of the line in the email message that indicates the news category. This line must be the first or second line of the messageString input
subheader_identifierSubheader line identifier: The starting characters of the line in the email message where you can add the text for the tt_news subheader field. This line must be the first or second line of the messageString input
default_categorydefault ttnews category: Title or ID of default ttnews category for news items created by this extension, used if no (valid) category has been set in email messageString input
news_cruser_idBackend user ID: ID of the backend user to whom the created news items will be assignedDatabase relationbe_users
hide_by_defaultHide new news items by default, so they will show on the website only after a backend user unhides themCheckbox, single
clearcachecmdClear cache of these pages: Comma separated list of page IDs where the created news records will be shown, of which the cache needs to be cleared for the new items to be shownString input, advanced
